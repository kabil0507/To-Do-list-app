# To-Do List App

A simple and interactive **To-Do List web application** built with **HTML, CSS, and JavaScript**. This app helps users manage daily tasks efficiently by allowing them to **add, mark as complete, and delete tasks**.

---

## Features

- **Add Tasks:** Users can enter new tasks and add them to the list.  
- **Mark as Complete:** Click on a task to mark it as done.  
- **Delete Tasks:** Remove tasks when they are completed or no longer needed.  
- **Responsive Design:** Works on both desktop and mobile devices.  

---

## Technologies Used

- **HTML** – Structure of the application  
- **CSS** – Styling and layout  
- **JavaScript** – Functionality and interactivity
